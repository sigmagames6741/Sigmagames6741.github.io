# Pizza Edition Game Website

A slick, clean gaming website featuring 500+ unblocked browser games from multiple collections, themed around pizza!

## Features

- **500+ Games**: Access to a vast library of unblocked browser games from various sources including Selenite, Chicken King's Vault, and UGS collections.
- **Pizza Theme**: Enjoy a warm, appetizing pizza-themed design with tomato red and cheese yellow accents.
- **Search Functionality**: Easily find your favorite games with the built-in search feature.
- **Settings**: Customize your experience with theme options, sound settings, and more.
- **Responsive Design**: Play on any device with a fully responsive layout.
- **Fullscreen Mode**: Immerse yourself in games with fullscreen support.

## How to Run

1. Open `index.html` in your web browser.
2. Browse the game library or use the search bar to find games.
3. Click on a game card to start playing.
4. Use the settings button (⚙️) to customize your experience.

## Game Sources

This website aggregates games from:
- [Selenite Collection](https://github.com/selenite-cc/selenite-old)
- [Chicken King's Vault](https://github.com/WanoCapy/ChickenKingsVault)
- [UGS Single File](https://github.com/bubbls/ugs-singlefile)
